# User Engagement Analytics
### Аналітика активності користувачів

This project collects and analyzes data on:
- Account creation dynamics
- User activity via emails (sending, opening, clicking)
- Behavior categories: sending intervals, account verification, subscription status
- Cross-country comparisons to identify key markets and segment users

Цей проєкт збирає та аналізує дані про:
- Динаміку створення акаунтів
- Активність користувачів за листами (відправлення, відкриття, переходи)
- Поведінкові категорії: інтервали відправлення, верифікація акаунтів, статус підписки
- Порівняння активності між країнами для визначення ключових ринків та сегментації користувачів
